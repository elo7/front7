O evento

O Front7 é um evento que nasceu dos desenvolvedores do Elo7 com o intuito de propagar conhecimentos e novas técnicas para o desenvolvimento front-end.

Agenda

  1. Abertura (18h30)

    Recepção

  2. Vida mais fácil com Chrome DevTools - parte 1 (18h45)

    Chrome DevTools: venha melhorar suas habilidades de front-end (design e produto também podem participar). Essa é uma ferramenta essencial para quem trabalha com web, vamos mostrar as abas Elements, Animation, Sources e como o Command Line pode te deixar mais produtivo.

  3. Coffee-break (19h30)

    Comidaaaa!

  4. Vida mais fácil com Chrome DevTools - parte 2 (19h45)

    Depois do coffee-break vamos falar sobre as abas Network para analisar a performance da sua aplicação, Acessibilidade, para saber como seu site está lidando com essa questão, Application para nos ajudar a desenvolver um site com PWA e Timeline/Profile para verificar a qualidade do seu código.
