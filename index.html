O evento

O Front7 é um evento que nasceu dos desenvolvedores do Elo7 com o intuito de propagar conhecimentos e novas técnicas para o desenvolvimento front-end.

Agenda

  1. Jogos: indo além do simples CSS! (18h30)

    Há quem pense que o CSS serve somente para aplicar estilos a determinados elementos, e realmente seu principal objetivo é esse! Mas alguém já pensou que seria possível capturar eventos, como um evento de clique por exemplo, e gerar algum tipo de animação com isso? E falar que com isso é possível criar um jogo? Neste talk explicarei como criar um jogo simples, sem uma linha de javascript, usando apenas HTML e CSS. Além de muitas de suas funcionalidades: pseudo-elementos, pseudo-classes, animations, entre outras.

  2. Coffee-break (19h30)

    Comidaaaa!

  3. Progressive Web Apps: o melhor da Web, appficada (19h45)

    A Web está evoluindo e correndo atrás das Apps. Muito do que só se fazia com código nativo hoje tem como na Web. Mas ainda faltava aquele passo final. A experiência de instalar algo, ter o ícone da home do usuário, usar offline, se integrar com a plataforma e rodar fora do browser. A Web chegou lá. Conheça as Progressive Web Apps instaláveis. Nessa palestra, vamos ver WebManifest, ServiceWorkers, PushNotifications e mais. Tudo para dar a melhor experiência da Web com um toque de App sob medida.
